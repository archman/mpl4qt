mpl4qt - Matplotlib Widget for PyQt5 and Qt5-designer¶

mpl4qt is the name of Python package (Python 3.x), which featuring matplotlib plotting widgets, as well as Qt5-designer integration, that is it possible for the user to develop PyQt GUI application by drag&drop way if data visualization widgets are required. Currently, fully support PyQt5 and Python 3.x.
