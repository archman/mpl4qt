For MatplotlibCurveWidget, the supported keyboard shotcuts are:

Prssed Key Function
a Turn on/off auto scale
g Turn on/off grid
l Turn on/off legend
m Turn on/off minor ticks
r Force refresh
s Change y-axis scale
t Tight layout on/off
? Show this message box
Esc Exit this message box

When running applications based on mpl4qt, press the above keys will instantly apply actions on the figure, press ? will pop up a dialog listing the short help message, just as the following image shows:
